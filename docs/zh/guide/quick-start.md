# 快速开始

## 从命令行启动

```bash
$ app-inspector -u YOUR-DEVICE-ID
```

关于如何获取设备 ID，请查看 <a href="./get-device-id.html">获取设备 ID</a> 部分。

## 打开界面

你的命令行将输出如下的文字：

> inspector start at: http://192.168.10.100:5678

然后在浏览器里面打开输出的链接：http://192.168.10.100:5678。推荐用 Chrome 浏览器。
